A set of tutorial videos accompany these files that will guide you step-by-step through setting up reservoir sedimentation models with two differnt approaches.  These videos are avialble (or will be shortly) on Youtube or on the Training page of the HEC website.

https://www.hec.usace.army.mil/training/default.aspx  (Webinars)

Data Files: Includes the project and geometry files that the tutorial starts with.

Reservoir Data.xlsx: Includes the data required to build the model.

Solution Files: Includes the completed model.
